# Hackapizza Chainlit Demo

Console chat-first per mostrare orchestrazione multi-agent e validazione vincoli.

## Uso rapido

- Scrivi una query direttamente in chat.
- Usa `/help` per vedere tutti i comandi disponibili.
- Modalita consigliata offline: `/mode structured` + `EMBEDDING_MODE=mock`.

## Comandi principali

- `/mode structured` oppure `/mode agent`
- `/constraints senza glutine, vegano`
- `/candidates Pizza Nebulosa; Insalata Orbitale`
- `/confidence 0.6`
- `/topk 4`
- `/evidence_limit 8`
- `/timeout 25`
- `/settings`
